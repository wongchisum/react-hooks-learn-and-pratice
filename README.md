# React Hooks学习实践

## 项目说明

使用vite脚手架创建项目，主要使用React Hooks与Typescript。

通过简单脚本实现约定式路由的创建（不支持嵌套，仅支持解析src/pages表层并生成路由配置）。

应用TS和React Hooks进行一些自定义Hooks的学习和实现。hooks主要来源于**usehooks**以及**ahooks**

## 项目运行

```
yarn

yarn start
```
